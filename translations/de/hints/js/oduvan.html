Beginnen wir mit einer einfachen Funktion, die erkennt, ob dies ein eindeutiges Element in einer gegebenen Liste ist.

Für den Moment habe ich ein paar Versionen für Sie zum Ausprobieren. Die erste ist länger: 

function isUnique(arr: number[], el: number): boolean {

    var hasOne = false;
    for (let item of arr) {
        if (item === el) {
            if (hasOne) {
                return false;
            } else {
                hasOne = true;
            }
        }
    }
    return true;
}

Die zweite ist kurz, aber vielleicht etwas schwieriger zu verstehen, wenn Sie neu in JS und ES6 sind. 

function isUnique(arr: number[], el: number): boolean {

    return arr.filter(a => a === el).length === 1
}

Wie kann ich damit diese Aufgabe lösen?

Stellen Sie sich das so vor: Sie müssen alle Elemente eines gegebenen Arrays durchgehen und ein neues Array mit allen eindeutigen Elementen erstellen. Auch hier zeige ich Ihnen zwei Wege zur Lösung der Aufgabe.

Eine lange:

function nonUniqueElements(data: number[]): number[] {

    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}

Und eine kurze:

function nonUniqueElements(data: number[]): number[] {

    return data.filter(item =>!isUnique(data, item))
}

Wie sollte also meine endgültige Lösung aussehen?
