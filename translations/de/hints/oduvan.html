Sicher. Hier ist, was Sie wissen müssen.

Um eine leere Liste zu initialisieren, tun Sie dies:

new_list = []

Um ein Element zur Liste hinzuzufügen, tun Sie dies:

new_list.append(item)

Um die Liste zurückzugeben, tun Sie dies:

return new_list

Wie sollte meine endgültige Lösung aussehen?

Sie müssen Ihren Code mit Hilfe von Typhinweisen in einen Funktionsausdruck verpacken. Da der Ausgabewert ein beliebiger Iterator sein kann, müssen Sie diesen Typ vor der Funktionsdefinition importieren.

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item)> 1:
            result.append(item)

    return result
