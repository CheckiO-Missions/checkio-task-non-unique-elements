Du erhältst eine nicht-leere Liste von Ganzzahlen (X). In dieser Aufgabe, sollst Du eine Liste bestehend nur aus den nicht-einmaligen Elementen daraus zurückgeben. Dazu musst Du alle einmaligen Elemente (Elemente welche in einer gegebenen Liste nur genau einmal vorkommen) entfernen. Zum Lösen dieser Aufgabe darfst Du die Reihenfolge der Liste nicht verändern. Beispiel: [1, 2, 3, 1, 3] 1 und 3 sind nicht-einmalige Elemente und das Resultat lautet [1, 3, 1, 3].

Eingabe: Eine Liste von Ganzzahlen.

Ausgabe: An iterable of integers.
