空でない整数のリスト(X)を与えられます。 このタスクでは、重複している要素だけでできているリストを返さなければいけません。 そのために、全てのユニークな要素(与えられたリストの中に１つだけ含まれる要素)を削除する必要があるでしょう。 このタスクを解くとき、要素の並びを変更してはいけません。 例: [1, 2, 3, 1, 3] 1と3は非ユニークな要素で、結果は[1, 3, 1, 3]になります。

入力: 整数のリスト

出力: 整数のイテラブル

どのように使われるか: このミッションは、配列の操作法を理解するのに役に立ちます。これは、より複雑なタスクを解決するための基礎になります。 このミッションの考え方は、簡単に実世界のタスクに一般化できます。 例えば: 低頻度の要素(ノイズ)を取り除くことで統計を明確にする必要があるときに使えます。

事前条件:
