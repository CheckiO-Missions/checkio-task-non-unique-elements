Oczywiście, oto co musi Pan wiedzieć.

Aby zainicjować pustą listę, proszę zrobić to:

new_list = []

Aby dodać element do listy proszę wykonać to:

new_list.append(item)

Aby zwrócić listę, proszę użyć tego:

return new_list

Tak więc poprawna odpowiedź może wyglądać następująco:

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item)> 1:
            result.append(item)

    return result

Zalecam również zapoznanie się ze wszystkimi dostępnymi list methods, aby wiedzieć, jakie inne metody mogą być potrzebne w przyszłości.
