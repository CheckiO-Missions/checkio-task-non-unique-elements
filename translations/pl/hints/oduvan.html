Musi Pan sprawdzić każdy element na podanej liście, czy pojawia się na niej więcej niż raz. Jeśli tak, proszę dodać ten element do nowej listy i zwrócić tę listę na końcu funkcji. Można więc użyć metody listy list.count(), która zwraca liczbę wystąpień x na liście.

Oto przykład: 

a = [1, 2, 2, 3, 4, 5, 6, 3, 2]
print(a.count(3))

Wyjście:

2

Proszę pomóc mi zastąpić komentarze.

Oczywiście, oto co musi Pan wiedzieć.

Aby zainicjować pustą listę, proszę zrobić to:

new_list = []

Aby dodać element do listy proszę wykonać to:

new_list.append(item)

Aby zwrócić listę proszę użyć tego:

return new_list

Jak powinno wyglądać moje ostateczne rozwiązanie?

Proszę opakować swój kod w wyrażenie funkcji używając podpowiedzi typu. Ponieważ wartością wyjściową może być dowolny Iterator, należy zaimportować ten typ przed definicją funkcji.

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item)> 1:
            result.append(item)

    return result

Zalecam również zapoznanie się ze wszystkimi dostępnymi list methods, aby wiedzieć, jakie inne metody mogą być potrzebne w przyszłości.

Czy mogę w jakiś sposób poprawić to rozwiązanie?

Zawsze można coś poprawić. Proszę spojrzeć na stronę Generators for Python.

Ok. Jak mogę tego użyć?

Oto prostsza wersja:

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    return (a for a in data if data.count(a)> 1)

Jest krótsza i jeszcze łatwiejsza do odczytania.
