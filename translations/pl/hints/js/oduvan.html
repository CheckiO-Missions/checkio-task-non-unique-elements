Proszę dać mi wskazówkę, jak rozwiązać tę misję.

Cóż, zacznijmy od prostej funkcji, która rozpoznaje, czy jest to unikalny element na danej liście.

Na razie mam kilka wersji do wypróbowania. Pierwsza jest dłuższa: 

function isUnique(arr: number[], el: number): boolean {

    var hasOne = false;
    for (let item of arr) {
        if (item === el) {
            if (hasOne) {
                return false;
            } else {
                hasOne = true;
            }
        }
    }
    return true;
}

Druga jest krótsza, ale może być trudniejsza do zrozumienia, jeśli są Państwo nowicjuszami w JS i ES6. 

function isUnique(arr: number[], el: number): boolean {

    return arr.filter(a => a === el).length === 1
}

Jak mogę to wykorzystać do rozwiązania tej misji?

Proszę pomyśleć w ten sposób: trzeba przejść przez wszystkie elementy danej tablicy i utworzyć tablicę wiadomości wypełnioną wszystkimi unikalnymi elementami. Ponownie, pokażę Państwu dwa sposoby rozwiązania tego zadania.

Długi:

function nonUniqueElements(data: number[]): number[] {

    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}

I krótki:

function nonUniqueElements(data: number[]): number[] {

    return data.filter(item =>!isUnique(data, item))
}

Jak powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy użyć obu funkcji:

    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}
