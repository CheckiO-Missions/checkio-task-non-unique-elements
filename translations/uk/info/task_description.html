Заданий непустий масив цілих чисел (Х). В цій задачі вам потрібно повернути масив, який складається тільки з неунікальних елементів даного масиву. Для цього необхідно видалити всі унікальні елементи (тобто ті, які наявні в масиві тільки один раз). Наприклад: [1, 2, 3, 1, 3], де 1 і 3 неунікальні елементи і результатом буде [1, 3, 1, 3].

Вхідні дані: Список (list) цілих чисел (int).

Вихідні дані: Iтератор (an iterable) цілих чисел (int).

Як це використовується: Ця задача допоможе вам зрозуміти, як маніпулювати з масивами. Також ця ідея може бути легко узагальнена для реальних задач. Для прикладу: якщо вам необхідно очистити статистику від елементів, які рідко зустрічаються (від шуму).
