Ти можеш використати метод списку "count", який повертає кількість конкретного значення Х у списку.

Ось наприклад:

>>> a = [1,2,2,3,4,5,6,3,2]
>>> a.count(3)
2
Як мені це використати для проходження місії?

Давай міркувати так: тобі потрібно пройти по всіх елементах вхідного списку і створити новий список із не унікальними елементами.

def checkio(data):
    # тут ти маєш створити новий порожній список
    for item in data:
        if data.count(item) > 1:
            # додай item у новий список, який ти створив раніше
    # поверни список, який ти створив на початку

Все, що тобі лишилося, це замінити коментарі кодом.

Мені потрібна допомога із заміною коментарів на код.

Звісно. Ось що тобі потрібно знати.

Щоб створити новий список, зроби наступне:

new_list = []

Щоб додати елемент до списку, зроби:

new_list.append(item)

Щоб повернути готовий список, використай:

return new_list

Тож остаточний код матиме вигляд:

def checkio(data):
    result = []
    for item in data:
        if data.count(item) > 1:
            result.append(item)
    return result

Я також раджу тобі поглянути на всі методи списків, адже вони тобі точно знадобляться у майбутньому.

Чи можна якось вдосконалити розв'язок?
Завжди є можливість вдосконалити рішення!. Ти можеш поглянути на Генератори у Python.
Так. І як це тут застосувати?

Просто заміни код

def checkio(data):
    result = []
    for item in data:
        if data.count(item) > 1:
            result.append(item)
    return result

наступним:

def checkio(data):
    return (a for a in data if data.count(a) > 1)

Так буде коротше і зрозуміліше.
