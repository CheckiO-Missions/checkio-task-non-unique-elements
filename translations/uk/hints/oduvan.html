Підкажіть, будь ласка, як вирішити цю місію.

Вам потрібно перевірити кожен елемент у заданому списку, чи не зустрічається він у ньому більше одного разу. Якщо так, то додати цей елемент до нового списку і повернути цей список в кінці функції. Таким чином, ви можете використовувати метод списку list.count(), який повертає кількість разів, коли x з'являється у списку.

Ось приклад: 

a = [1, 2, 2, 3, 4, 5, 6, 3, 2]
print(a.count(3))

Виведення:

2

Як я можу використати це для розв'язання цієї місії?

Все, що вам потрібно зробити, це замінити коментарі на відповідний код на python. 

# init empty list
for item in data:
    if data.count(item)> 1:
        # add the current item to the new list you initialized before
# return the list

Будь ласка, допоможіть мені замінити коментарі.

Звичайно. Ось що вам потрібно знати.

Щоб ініціалізувати порожній список, зробіть це:

new_list = []

Щоб додати елемент до списку, зробіть це:

new_list.append(item)

Щоб повернути список, скористайтеся цим:

return new_list

Як має виглядати моє кінцеве рішення?

Вам потрібно обернути ваш код у вираз функції, використовуючи підказки типів. Оскільки вихідним значенням може бути будь-який ітератор, вам потрібно імпортувати цей тип перед визначенням функції.

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item)> 1:
            result.append(item)

    return result

Гаразд. Як я можу ним користуватися?

Ось простіша версія:

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    return (a for a in data if data.count(a)> 1)

Вона коротша і ще легша для читання.
