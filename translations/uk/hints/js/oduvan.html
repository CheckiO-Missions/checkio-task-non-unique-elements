Підкажіть, будь ласка, як вирішити цю місію.

Ну, почнемо з простої функції, яка розпізнає, чи це унікальний елемент у заданому списку.

Наразі у мене є кілька варіантів, які ви можете спробувати. Перший варіант довший: 

function isUnique(arr: number[], el: number): boolean {

    var hasOne = false;
    for (let item of arr) {
        if (item === el) {
            if (hasOne) {
                return false;
            } else {
                hasOne = true;
            }
        }
    }
    return true;
}

Другий варіант коротший, але може бути складнішим для розуміння, якщо ви новачок в JS та ES6. 

function isUnique(arr: number[], el: number): boolean {

    return arr.filter(a => a === el).length === 1
}

Як я можу використати це для вирішення цієї місії?

Подумайте про це так: вам потрібно перебрати всі елементи заданого масиву і створити масив новин, заповнений всіма унікальними елементами. Знову ж таки, я покажу вам два способи вирішення цієї задачі.

Довгий:

function nonUniqueElements(data: number[]): number[] {

    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}

І короткий:

function nonUniqueElements(data: number[]): number[] {

    return data.filter(item =>!isUnique(data, item))
}

Отже, як має виглядати моє остаточне рішення?

Вам просто потрібно використовувати обидві функції: 

function nonUniqueElements(data: number[]): number[] {

    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}
function isUnique(arr: number[], el: number): boolean {

    var hasOne = false;
    for (let item of arr) {
        if (item === el) {
            if (hasOne) {
                return false;
            } else {
                hasOne = true;
            }
        }
    }
    return true;
}

function nonUniqueElements(data: number[]): number[] {

    return data.filter(item =>!isUnique(data, item))
}
