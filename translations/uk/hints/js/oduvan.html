Хотів би отримати підказку, як розвязати цю міссію.

Що ж, почнемо із простої функції, яка перевіряє, запитуваний елемент унікальний в листі.

Поки що маю для тебе дві версії розв'язку. Перша довша:

function isUnique(arr, el) {
    var hasOne = false;
    for (let item of arr) {
        if (item === el) {
            if (hasOne) {
                return false;
            } else {
                hasOne = true;
            }
        }
    }
    return true;
}

Друга коротша, але може бути важкою для розуміння, якщо ти новачок в JS і ES6

function isUnique(arr, el) {
    return arr.filter(a => a === el).length === 1
}
Як мені використати це тут?

Подумай таким чином: тобі потрібно пройти всі елементи запропонованого списку і створити новий список з не унікальними елементами. І знову, я покажу тобі два способи.

Це довший:

function nonUniqueElements(data) {
    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}

І короткий:

function nonUniqueElements(data) {
    return data.filter(item => !isUnique(data, item))
}
