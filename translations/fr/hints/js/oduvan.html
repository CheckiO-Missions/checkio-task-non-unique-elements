Veuillez me donner un indice sur la façon de résoudre cette mission.

Commençons par une fonction simple qui reconnaît s'il s'agit d'un élément unique dans une liste donnée.

Pour l'instant, j'ai quelques versions à vous proposer. La première est plus longue: 

function isUnique(arr: number[], el: number): boolean {

    var hasOne = false;
    for (let item of arr) {
        if (item === el) {
            if (hasOne) {
                return false;
            } else {
                hasOne = true;
            }
        }
    }
    return true;
}

La seconde est courte mais peut-être plus difficile à comprendre si vous êtes novice en JS et ES6. 

function isUnique(arr: number[], el: number): boolean {

    return arr.filter(a => a === el).length === 1
}

Comment puis-je utiliser ceci pour résoudre cette mission?

Pensez-y de la manière suivante: vous devez passer en revue tous les éléments d'un tableau donné et créer un nouveau tableau contenant tous les éléments uniques. Là encore, je vous montrerai deux façons de résoudre ce problème.

Une longue:

function nonUniqueElements(data: number[]): number[] {

    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}

Et une courte:

function nonUniqueElements(data: number[]): number[] {

    return data.filter(item =>!isUnique(data, item))
}

Alors, à quoi devrait ressembler ma solution finale?

Il vous suffit d'utiliser les deux fonctions: 

function nonUniqueElements(data: number[]): number[] {

    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}
function isUnique(arr: number[], el: number): boolean {

    var hasOne = false;
    for (let item of arr) {
        if (item === el) {
            if (hasOne) {
                return false;
            } else {
                hasOne = true;
            }
        }
    }

}

}
