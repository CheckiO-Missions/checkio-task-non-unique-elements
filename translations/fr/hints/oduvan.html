Pour initialiser une liste vide, faites ceci:

new_list = []

Pour ajouter un élément à la liste, faites ceci:

new_list.append(item)

Pour retourner la liste, utilisez ceci:

return new_list

À quoi devrait ressembler ma solution finale?

Vous devez envelopper votre code dans une expression de fonction en utilisant des indications de type. Comme la valeur de sortie peut être n'importe quel itérateur, vous devez importer ce type avant la définition de la fonction.

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item)> 1:
            result.append(item)

    return result
