On vous propose un tableau d'entiers. Pour cette tache, vous devez retourner un tableau qui contient seulement les éléments non uniques dans cette liste. Pour ce faire vous devez supprimer tous les éléments uniques (éléments qui ne sont présents dans la liste qu'une seule fois). Le tableau d'entiers de résultat ne doit pas changer l'ordre du Le tableau d'entiers de départ. Exemple: [1, 2, 3, 1, 3] 1 et 3 sont des éléments non uniques, le résultat sera donc [1, 3, 1, 3].

Intérêt de la mission Cette mission vous aidera à comprendre comment manipuler des tableaux,ce qui est une base pour résoudre des tâches plus complexes. Ce concept peut être facilement transposé dans la vie réelle. Par exemple: si vous avez besoin de modifier des statistiques en enlevant les élèments qui ne reviennent pas souvent.
