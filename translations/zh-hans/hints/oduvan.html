您需要检查给定列表中的每个元素是否在其中出现过不止一次。如果是，则将该元素添加到一个新列表中，并在函数结束时返回该列表。因此，您可以使用列表方法list.count() 来返回x 在列表中出现的次数。

下面是一个示例： 

a = [1, 2, 2, 3, 4, 5, 6, 3, 2]
print(a.count(3))

输出：

2

我的最终解决方案应该是怎样的？

您需要使用类型提示将代码封装为函数表达式。由于输出值可以是任何迭代器，因此需要在函数定义前导入该类型。

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item)> 1:
            result.append(item)

    return result
