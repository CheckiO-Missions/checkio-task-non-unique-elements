Sure. Here is what you need to know.

To initialize an empty list do this:

new_list = []

To add an item to the list do this:

new_list.append(item)

To return the list use this:

return new_list

So the correct answer might look like this:

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item) > 1:
            result.append(item)

    return result
