How my final solution should look like?

You need to wrap your code into function expression using type hints. Since output value may be any Iterator, you need to import this type before fuction definition.

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    result = []
    for item in data:
        if data.count(item) > 1:
            result.append(item)

    return result

Here is a simpler version:

from collections.abc import Iterator


def checkio(data: list[int]) -> Iterator[int]:

    return (a for a in data if data.count(a) > 1)
