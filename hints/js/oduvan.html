Well, let’s starting with a simple function that recognizes if this is a unique element in a given list.

For now I have a couple versions for you to try. First one is longer: 

function isUnique(arr: number[], el: number): boolean {

    var hasOne = false;
    for (let item of arr) {
        if (item === el) {
            if (hasOne) {
                return false;
            } else {
                hasOne = true;
            }
        }
    }
    return true;
}

The second one is short but maybe harder to understand if you are new to JS and ES6. 

function isUnique(arr: number[], el: number): boolean {

    return arr.filter(a => a === el).length === 1
}

Think about this way: you need to go through all elements of a given array and create a news array filled with all unique elements. Again, I’ll show you two ways to solve it.

A long one:

function nonUniqueElements(data: number[]): number[] {

    const new_result = []
    for (let item of data) {
        if (!isUnique(data, item)) {
            new_result.push(item)
        }
    }
    return new_result;
}

And a short one:

function nonUniqueElements(data: number[]): number[] {

    return data.filter(item => !isUnique(data, item))
}
